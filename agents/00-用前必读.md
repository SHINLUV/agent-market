# 用前必读：10 分钟配好你的 AI 运营助手

## 你需要准备
1. 一台电脑（Win 或 Mac）
2. 10 分钟时间
3. 手机号（注册 DeepSeek 用）

## 第一步：装 Cherry Studio
1. 打开官网：https://www.cherry-ai.com/download
2. 点 Windows 版本下载
3. 双击安装，一路点"下一步"
4. 桌面出现樱桃图标，打开它

## 第二步：注册 DeepSeek
1. 打开：https://platform.deepseek.com
2. 手机号注册，送 500 万 token 免费额度
3. 注册完点「API Keys」→「创建 API Key」
4. 复制那串 sk- 开头的密钥，保存好

## 第三步：配 Cherry Studio
1. 打开 Cherry Studio
2. 左下角点「设置」⚙️
3. 点「模型服务」→ 找到「DeepSeek」
4. 粘贴刚才的 API Key → 点「连接」
5. 点「助手」→「新建助手」
6. 起个名字（比如"数据分析师"）
7. 在「系统提示词」框里粘贴 Agent 的 Prompt 模板
8. 保存

## 第四步：开始用
1. 在 Cherry Studio 左侧点刚创建的助手
2. 把数据粘贴进去 → 回车
3. AI 开始帮你分析

## 常见问题

**Q：为什么不用 ChatGPT/豆包/Kimi？**
都可以用。但这个模板是针对 DeepSeek 调过的，用 DeepSeek 效果最好。因为 DeepSeek 的中文理解能力超强，而且便宜。

**Q：我的数据安全吗？**
DeepSeek 不会拿你的数据去训练模型。Cherry Studio 数据只存在你电脑上。

**Q：我不会写提示词，能用吗？**
模板已经写好了，你只填数字，不需要自己写任何 prompt。

**Q：效果不好怎么办？**
多给 AI 一点信息。比如不只是"访客 1200"，而是"访客 1200，昨天周三，下雨"。越具体，分析越准。
